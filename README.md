# 💸 PocketFlow – Moderná správa financií
![Status](https://img.shields.io/badge/status-in%20development-orange)
  - projekt je stále vo fáze implementácie

PocketFlow je webová aplikácia postavená na najnovších technológiách pre efektívne sledovanie osobných financií. Projekt demonštruje prácu s autentifikáciou, databázou a jednoduchým používateľským rozhraním.

## 🚀 Technológie

Projekt využíva moderné nástroje, ktoré zabezpečujú rýchlosť, bezpečnosť a škálovateľnosť:

- **Frontend:** [Next.js 15](https://nextjs.org/) (App Router)
- **Jazyk:** [TypeScript](https://www.typescriptlang.org/) – pre typovú bezpečnosť
- **Databáza:** [Supabase](https://supabase.com/) – využitie PostgreSQL
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) – pre responzívny dizajn
- **Komponenty:** [shadcn/ui](https://ui.shadcn.com/) – profesionálne prístupné UI komponenty.
- **Deployment:** [Vercel](https://vercel.com/) – optimalizované pre Next.js prostredie
